# 🚀 微信小程序自动化 - 快速开始指南

## 第一步：开启开发者工具服务端口

1. 打开**微信开发者工具**
2. 点击右上角 **设置** 图标
3. 选择 **安全设置**
4. 勾选 **服务端口** 选项（默认端口 9420）

![服务端口设置](https://developers.weixin.qq.com/miniprogram/dev/devtools/auto/image/auto_settings.png)

## 第二步：安装依赖

在小程序项目目录下执行：

```powershell
# 进入小程序目录
cd "D:\Desktop\微信机器人Flask\微信Flask-小程序"

# 安装自动化依赖
npm install miniprogram-automator --save-dev
```

或者直接在项目根目录运行：
```powershell
npm install
```

## 第三步：运行自动化脚本

### 方式 A：使用 npm 脚本（推荐）

```powershell
# 基础连接测试
npm run auto

# 运行示例集合
npm run auto:examples
```

### 方式 B：直接运行 Node.js

```powershell
# 基础连接
node automation/setup.js

# 示例集合
node automation/examples.js
```

## 配置说明

如果需要修改开发者工具路径，编辑 `automation/setup.js`：

```javascript
const config = {
  // Windows 路径（根据实际安装位置修改）
  cliPath: 'C:\\Program Files (x86)\\Tencent\\微信web开发者工具\\cli.bat',
  
  // 项目路径（自动获取，无需修改）
  projectPath: path.join(__dirname, '..'),
  
  // 小程序 appid
  appid: 'wx215d5c0f12d30adf',
};
```

## 常见的开发者工具安装路径

### Windows
- 默认路径: `C:\\Program Files (x86)\\Tencent\\微信web开发者工具\\cli.bat`
- 自定义安装: 查看你的安装目录

### macOS
- 默认路径: `/Applications/wechatwebdevtools.app/Contents/MacOS/cli`

## 常见问题排查

### ❌ 连接失败：ECONNREFUSED

**原因:** 开发者工具服务端口未开启

**解决:**
1. 打开微信开发者工具
2. 设置 -> 安全设置 -> 勾选"服务端口"
3. 重启开发者工具

### ❌ cli.bat not found

**原因:** 开发者工具路径配置错误

**解决:**
1. 找到开发者工具的实际安装路径
2. 修改 `automation/setup.js` 中的 `cliPath` 配置

### ❌ 项目未加载

**解决:**
- 使用 `connect` 方法前，先在开发者工具中打开项目
- 或使用 `launch` 方法自动启动（需配置正确的 cliPath）
