**修改内容**:
```json
"requiredPrivateInfos": [
  "getLocation",

  "scope.userLocation": {
    "desc": "您的位置信息将用于提供基于地理位置的服务"
```

### 3. pages/login/index.js（登录逻辑）

**修改位置**: 
1. data 数据新增 `agreed: false`
2. 登录方法前增加协议同意检查
3. 新增跳转方法

**新增/修改内容**:
```javascript
data: {
  agreed: false,  // 是否同意协议
  // ... 其他数据
},

// 协议同意状态变更
onAgreeChange(e) {
  const agreed = e.detail.value.includes('agree');
  this.setData({ agreed });
},

// 跳转到用户协议
goToTerms() {
  wx.navigateTo({
    url: '/pages/terms/index'
  });
},

// 跳转到隐私政策
goToPrivacy() {
  wx.navigateTo({
    url: '/pages/privacy/index'
  });
},

// 修改登录方法，增加协议检查
onWechatLogin() {
  if (!this.data.agreed) {
    wx.showModal({
      title: '提示',
      content: '请先阅读并同意《用户协议》和《隐私政策》',
      showCancel: false
    });
    return;
  }
  // ... 原有登录逻辑
}
```

**修改理由**: 
- 确保用户在登录前已同意协议
- 提供便捷的协议查看入口
